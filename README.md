# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/raj2802abhishek/abhishek-portfolio.git
```

Navigate to project:

```bash
cd abhishek-portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```
